Hash (v 1.0.11)
By Sriranga Veeraraghavan <ranga@calalum.org>

About:

Hash is a MacOSX application to generate and verify the hash, crc, or checksum
of a file.  Hash supports CRC32, Unix-style checksums, MD5, SHA1, SHA256,
SHA512, RMD160, RMD320, Whirlpool, SHA3 256, SHA3 512, BLAKE2B 256, BLAKE2B
512, Skein 256-256, Skein 512-256, and Skein 512-512.  
 
Urls:

Binary - https://grommit.com/~ranga/pgms/hash/Hash-1.0.11.zip
Source - https://github.com/srirangav/Hash

Installation:

Copy Hash.app to /Applications (or wherever you prefer)
